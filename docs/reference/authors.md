## How to contribute

mzPeak is developed in the open. The most valuable contribution right now is
**careful review of this specification** by readers with fresh eyes, reporting
anything unclear or missing.

- File issues or suggestions:
  [github.com/HUPO-PSI/mzPeak-specification/issues](https://github.com/HUPO-PSI/mzPeak-specification/issues)
- Reference implementation (Rust):
  [github.com/HUPO-PSI/mzPeak](https://github.com/HUPO-PSI/mzPeak)
